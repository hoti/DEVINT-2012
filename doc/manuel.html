  • attention, le jeu attend sans rien dire, alors que c'est à vous de jouer
  • il faut une résolution de 1024 par 768 pixels au moins

Règles du jeu (contexte, objectif, déroulement)

La Bataille Navale est un jeu de société dans lequel deux joueurs doivent placer des bateaux sur une grille qui représente un champ de bataille navale, et ensuite lancer des torpilles sur l'adversaire pour le détruire.
Chaque joueur lance une torpille à son tour. Pour couler un navire adverse, il faut atteindre toutes les cases qu'il occupe. A chaque fois qu'une partie d'un navire est atteinte, on dit que ce navire est touché, quand toutes les parties sont atteintes, ce navire est coulé.
La partie est terminée quand la flotte de l'un des deux joueurs est totalement coulée.

Ici, le joueur de droite est l'ordinateur, alors que la grille de gauche montre la flotte du joueur humain.

Comment jouer (commandes, aides)

Dés qu'on lance le jeu, on a accès à des explications suffisantes. Le menu d'accueil contient quatre rubriques, dont une rubrique d'aide, où l'on navigue avec les flèches haut et bas. On utilise la touche espace pour valider, on peut utiliser la touche Echap pour terminer :

  • Nouvelle partie (touche Entrée)
  • Aide (touche F1)
  • Configuration
  • Quitter (touche Echap)

Public concerné
